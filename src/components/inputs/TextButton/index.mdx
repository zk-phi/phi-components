#### Supported properties

<Markdown>
{`
| Property     | Type                  | Default    | Description                        |
|--------------|-----------------------|------------|------------------------------------|
| \`onClick\`  | \`() => void\`        | (required) | A handler function called on click |
| \`danger\`   | \`boolean\`           | \`false\`  | Toggle danger color variant        |
| \`children\` | \`ComponentChildren\` |            | Icon elementent                    |
`}
</Markdown>
