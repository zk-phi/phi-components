### Preact

``` jsx
return (
  <Icon icon="back" />
);
```

### Custom Element

``` html
<phi-icon icon="back" />
```

### Vanilla JS x CSS

Unsupported.

## Reference
### Preact
#### Properties

| Property | Type     | Default    | Description       |
|----------|----------|------------|-------------------|
| `icon`   | `string` | (required) | Icon name to show |

### Custom Element
#### Props / Attrs

| Slot   | Category  | Type     | Default |
|--------|-----------|----------|---------|
